# Agent
A Swift library for HTTP and WebSocket

## Features
- [x] HTTP GET PUT POST DELETE Methods
- [ ] WebSocket
- [ ] HTTP Fake Request
- [ ] HTTP File Upload Download Requests
- [ ] HTTP File Request Group
- [ ] HTTP HEAD PATCH Methods
- [ ] Generic Codable Request
- [ ] Cached Image Request

# Agent
A Swift library for HTTP and WebSocket

## Features
- [x] HTTP GET PUT POST DELETE Request
- [ ] WebSocket Request
- [ ] Fake Request
- [ ] HTTP File Upload Download Request
- [ ] HTTP HEAD PATCH Request
- [ ] HTTP Custom Request Timeout
- [ ] HTTP File Request Group
